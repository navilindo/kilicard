Njama

It is a game played on Banga board by bringing together four cards with the same color into four adjacent squares

How to play Njama?

  • The game is played by two players/teams.
  • Start a new game by shuffling the cards then systematically distribute them throughout the board by placing them(frontside up) to each plane square in clockwise/ anticlockwise direction. .(image of Banga Board to show direction of distribution that starts infront of green square)
  • The following hints should be put into consideration before the get-go
  • Every player should carefully look and examine the number and the values of the cards with the same color lying at his/her side
  • If the side of one player has cards with higher value than the cards (with the same color) at the opponent side he/she can pick any card with the same color from his/her opponent side and replace it with any other card from his/her side.
  • The value of the card is determined by the number of words/sentences Example, Consider the cards with red colour below :
  • The special cards have the value of 77 weight. (image no 29 to 32)
  • The special cards can also be used to pick a card from the opponent side but only once
  • If both sides have cards with the same values then the replacement of cards can not be done.
  • A payer can move his/her card one step to the adjacent plane square in any direction
  • The winner is the one who manages to bring together four cards with the same color into four adjacent squares(at the Peak Point) .(image of Banga Board to show presence of four cards with the same color together into four adjacent squares(at the Peak Point)
