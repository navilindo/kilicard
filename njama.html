    Baso Kotea Njama

Games > Njama

It is a game played on the Banga board by bringing together four cards with the same colour into four adjacent squares (Peak Point) by sliding them either toward vertical or horizontal direction at each player’s side.

How to play Njama?

The game has twelve (12) rules, the following are the rules of playing NJAMA.

  1. The game is played by two players/teams whereby the peak of Mount Kilimanjaro is the boundary for each player.
  2. Start a new game by shuffling the cards then systematically distribute them throughout the board by placing them (frontside up) to each plain square in clockwise/ anticlockwise direction.



    OR



  3. Every player should carefully look and examine the number and the values of the cards with the same colour lying at his/her side.



  4. If the side of one player has cards with higher value than the cards (with the same colour) at the opponent’s side, he/she can pick any card with the same colour from his/her opponent’s side and replace it with any other card from his/her side.
  5. The value of the card is determined by the number of Swahili-English word/ sentence pairs.
