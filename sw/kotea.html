    Baso Kotea Njama

Games > Kotea

It is a game played on the Banga Board by randomly picking up the cards then placed at the coloured squares by considering the similarity of the colour of particular card at each player’s side.



How to play Kotea?

The game has nine (9) rules; the following are the rules of playing KOTEA.

  1. This game can be played by two players/teams.
  2. Start a new game by shuffling the cards then systematically distribute them throughout the board by placing them (backside up) to each plain square in clockwise / anticlockwise direction.



    3. OR



  3. The first player will pick any card then place it on a coloured square (frontside up) which has similar colour to that of the card.



  4. The second player will also pick any card and place it on a coloured square (frontside up) which has similar colour to that of the card.



  5. Every player should use not more than 15 seconds on his turn until all cards are picked.

  6. A player will score one(1) point if he/she obtains four cards with the same colour, otherwise it is a zero (0) score.



  7. A player will score super a goal which is equal to seven (7) points if he/she obtains four special cards, otherwise it is a zero (0) score.
  8. The winner will be the one with the total higher score than the other.
  9. If both players have the same total scores the game is draw.
